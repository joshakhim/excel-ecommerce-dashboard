# E-Commerce Excel Dashboard

## Overview

This project focuses on cleaning, analyzing, and visualizing e-commerce sales data using Microsoft Excel.

## Project Workflow

* Cleaned messy raw data
* Removed unusable rows
* Standardized categories and dates
* Created Total Sales calculations
* Built Pivot Tables
* Designed an interactive dashboard using slicers and charts

## Dashboard Insights

* Sales by category
* Country performance
* Best-selling products
* Monthly revenue trends

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Data Cleaning Techniques

## Project Structure

* `data/` → raw and cleaned datasets
* `excel/` → final Excel dashboard workbook
* `images/` → dashboard screenshots
